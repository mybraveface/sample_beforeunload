チェックリスト

1. 変更・保存ボタンを押さずにタブを閉じる。 => 確認モーダルが表示されずにタブが閉じる。

2. 変更ボタンを押して「変更あり」状態でタブを閉じる => 確認モーダルが表示され、「このページを離れる（文言はブラウザにより異なる）」押下でタブが閉じる。

3. 変更ボタンを押して「変更あり」状態でタブを閉じる => 確認モーダルが表示され、「キャンセル」押下でタブを閉じる動作がキャンセルされる。

4. 変更ボタンを押した後に保存ボタンを押して「変更なし」状態でタブを閉じる => 確認モーダルが表示されずにタブが閉じる。

5. 変更ボタンを押した後に保存ボタンを押して、さらに変更ボタンを押して「変更あり」状態でタブを閉じる => 確認モーダルが表示され、「このページを離れる（文言はブラウザにより異なる）」押下でタブが閉じる。

6. 変更ボタンを押した後に保存ボタンを押して、さらに変更ボタンを押して「変更あり」状態でタブを閉じる => 確認モーダルが表示され、「キャンセル」押下でタブを閉じる動作がキャンセルされる。

7. 変更ボタンを押して「変更あり」状態でタブを閉じる => 確認モーダルが表示され、「キャンセル」押下でタブを閉じる動作がキャンセルされる。 => この動作を複数回繰り返し、「このページでこれ以上ダイアログボックスを生成しない（文言はブラウザにより異なる」が表示されたらチェックして「このページを離れる（文言はブラウザにより異なる）」押下でタブを閉じる。 => 「変更あり」状態でタブを閉じても確認モーダルが表示されなくなる。

8. 変更ボタンを押して「変更あり」状態でタブを閉じる => 確認モーダルが表示され、「キャンセル」押下でタブを閉じる動作がキャンセルされる。 => この動作を複数回繰り返し、「このページでこれ以上ダイアログボックスを生成しない（文言はブラウザにより異なる」が表示されたらチェックして「キャンセル」押下で確認ダイアログタブを閉じる。 => 「変更あり」状態でタブを閉じる際に確認モーダルが再表示される。

9. 7の操作後、ブラウザを再起動した場合、「変更あり」状態でタブを閉じる動作を行った場合、確認モーダルが再度表示されるようになる。

10. 確認モーダル表示中にもう一度タブを閉じる動作を行う => タブが閉じて確認モーダルのみ残る。
